📋 Sobre o projeto

Aplicativo desenvolvido no Microsoft Power Apps com o objetivo de gerenciar um salão de beleza de forma simples e eficiente.

O sistema permite organizar atendimentos, clientes e serviços, facilitando o dia a dia e melhorando o controle das informações.

---

🚀 Funcionalidades

- 📅 Agendamento de clientes
- 👥 Cadastro de clientes
- 💇‍♀️ Controle de serviços
- 💲 Consulta de preços
- 📊 Organização das informações

---

🛠️ Tecnologias utilizadas

- Microsoft Power Apps
- Power Platform

---

📦 Estrutura do projeto

Este repositório contém o arquivo exportado do Power Apps:

- "SalaoNovaBeleza.zip"

📌 O arquivo pode ser importado novamente no Power Apps para edição e uso.

---

▶️ Como usar

1. Acesse o Power Apps
2. Vá em Soluções
3. Clique em Importar solução
4. Selecione o arquivo ".zip" deste repositório
5. Aguarde a importação
